Movie Trailer Website

This project involved making a Movie Trailers website, following an understanding of HTML and CSS fundamentals and a file called fresh_tomatoes. The program provides a list of favourite movies, where users can click on each image to view its individual trailer.

  • a data structure was created to store the movies, including movie title, box art URL (or poster URL) and a YouTube link to the movie trailer
  • classes define favourite movies
  • a website is generated that displays these movies, with the assistance of fresh_tomatoes.py

Portfolio

This assignment involved making a repsonsive Portfolio webpage. A design mockup was provided by Udacity, and the design was recreated. The design includes:

  • own HTML and CSS files
  • personalised portfolio with custom colors, additional content and own images
  • responsive design with the assistance of the Bootstrap framework

A number of personalised media queries and breakpoints have been used to create an enhanced user experience.
